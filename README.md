# Waterformer

Course Project for the AIX 821 Spatial Computing course.

Steps to run:

```
pip install -r requirements.txt
```

```
streamlit run app.py
```

This will open the website locally. To see the hosted website, visit https://watersegmentation.streamlit.app.
